# 轻盘 · AI 空间防火墙

轻盘是一款面向桌面端的 AI 存储管理产品，目标不是简单删除文件，而是帮助用户看懂磁盘空间为什么增长、哪些应用正在持续占用空间，以及怎样安全地释放容量。

## 当前能力

- 深色与浅色主题切换
- 磁盘空间和应用占用概览
- 微信、剪映、Xcode、飞书等应用的空间分析
- 应用增长趋势与异常事件展示
- AI 清理方案交互原型
- 安全区、恢复和应用预算的产品界面

当前版本属于可交互的前端产品原型，扫描进度、磁盘数据和 AI 结论仍为演示数据。后续桌面版将通过本地执行层读取真实文件系统，并接入可切换的 AI 模型服务。

## 产品原则

- 默认只在本地扫描文件元数据
- AI 负责解释和生成建议，不直接删除文件
- 清理操作必须经过风险规则和用户确认
- 优先隔离后删除，并提供恢复记录
- 支持按应用设置空间预算和增长提醒

## 技术栈

- React 19
- Next.js 16
- TypeScript
- vinext / Vite
- Cloudflare Worker

## 本地运行

需要 Node.js `>=22.13.0`。

```bash
npm install
npm run dev
```

常用命令：

- `npm run dev`：启动本地开发环境
- `npm run build`：构建项目
- `npm test`：构建并运行页面渲染测试
- `npm run lint`：运行代码检查

## 项目状态

下一阶段计划：

1. 使用 Tauri 封装 macOS 桌面应用。
2. 接入只读的本地文件扫描能力。
3. 建立应用识别和安全清理规则。
4. 接入 Gemini、DeepSeek、OpenAI或本地模型。
5. 增加隔离区、恢复记录和真实清理能力。

本仓库不在客户端源码中保存任何 AI API Key。正式版本应通过安全的服务端代理调用云端模型，或将用户自有密钥保存在系统钥匙串中。
